# SchulCloudViewer

`scraper` ist ein Python3-Modul, welches die Aufgaben und dazugehörige Attribute von der SchulCloud Brandenburg lädt und im Terminal ausgibt. Die Instanz der SchulCloud kann in `scraper/vars.py` angepasst werden.

## Setup

1. Neues virtual environment im Ordner `env` einrichten und aktivieren: `python3 -m venv env; source ./env/bin/activate`
2. Erforderliche Module von PyPI installieren: `pip3 install -r requirements.txt`
3. Modul mit `python3 -m scraper` ausführen
